About Our Online Attendance Management System

Introduction

Welcome to my Online Attendance Management System! I'm Amulya Kumar Padhan, a computer science student specializing in artificial intelligence and web development. This project is a demonstration of my skills and expertise in building web-based applications that incorporate advanced technologies like computer vision and face recognition.

Project Overview

The Online Attendance Management System is designed to automate and streamline the process of marking attendance using face recognition technology. It combines traditional web development with cutting-edge machine learning to create a solution that is both efficient and user-friendly.

Key Features

  • Web-Based Platform: This system is accessible through a simple web interface, allowing users to mark attendance online.
  • Face Recognition: I used OpenCV and DeepFace to implement face detection and recognition, providing an automated way to mark attendance.
  • Secure and Efficient: The system ensures secure handling of user data while providing quick and accurate attendance recording.

Technologies Used

  • Django: This powerful web framework allowed me to build a robust backend for the system.
  • OpenCV: A versatile computer vision library used for face detection and image processing.
  • DeepFace: A popular face recognition library that leverages deep learning to identify and verify faces.
  • JavaScript: Used for client-side interactions, including capturing webcam images for face recognition.
  • Bootstrap: A frontend framework to create a responsive and modern user interface.
